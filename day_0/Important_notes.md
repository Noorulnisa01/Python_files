# ***Note 2: Objects and Function in Python***
## ***2.1 Object:***
> - A Command is written ***without ( )*** it is called as object.
> - For example: **df.shape**

## ***2.2 Function:***
> - A Command is written ***with ( )*** it is called as function.
> - For example: **df.head()**
